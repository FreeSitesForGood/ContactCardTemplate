# Contact Card Template

This is a minimalist, Carrd-style contact card template built with [Eleventy](https://www.11ty.dev/), [Tailwind CSS](https://tailwindcss.com/), [Nunjucks](https://mozilla.github.io/nunjucks/), and [FontAwesome](https://fontawesome.com/) for social icons. It's designed for nonprofits, community orgs, and mutual aid groups that need to get something online *fast* ‚Äî no fluff, no paywall, no gatekeeping.

## Features
- Single-page layout with name, email, phone, and optional social links
- Built for GitHub Pages ‚Äî auto-deploy on every push to `main`
- Customize with a single JSONC file (`site-config.jsonc`)
- Fully mobile-friendly and accessibility-minded
- Social icons via FontAwesome

## Getting Started
1. [Fork this repo](https://github.com/YOUR-ORG/contact-card-template)
2. Edit `site-config.jsonc` with your info
3. Commit to `main`
4. Site builds automatically via GitHub Actions and publishes to GitHub Pages
